## Задание 1 — Развернуть скиллы в проекте ✅📦

**Цель:** Установить скиллы из предыдущего занятия (Лекция 2) в свой проект.

### Что нужно сделать:
1. Открыть свой проект в Antigravity
2. Установить необходимые скиллы (глобально и локально)
3. Убедиться, что скиллы работают корректно

### Скиллы для установки:
- `find-skills` — поиск скиллов
- `frontend-design` — дизайн интерфейсов
- `canvas-design` — работа с Canvas
- `jspdf-export` — генерация PDF
- `web-design-guidelines` — веб-стандарты

> ⚠️ **Рекомендация:** Устанавливайте скиллы по одному. Не ставьте все разом — может зависнуть!

## Задание 2 — Создать репозиторий на GitHub 🗃️

**Цель:** Создать персональный репозиторий и залить туда свой проект.

### Пошаговая инструкция:

1. **Создать репозиторий** на GitHub:
   - Зайти на https://github.com
   - Нажать **New Repository**
   - Дать имя (латиницей)
   - Выбрать **Public** или **Private**
   - Нажать **Create Repository**
   - Скопировать ссылку

2. **Настроить Git** (если ещё не сделано):
   ```powershell
   git config --global user.name "Ваше Имя"
   git config --global user.email "email@example.com"
   ```

3. **Залить код** в репозиторий:
   ```powershell
   git clone https://github.com/ваш-аккаунт/имя-репо.git
   # скопировать файлы проекта в папку репо
   cd имя-репо
   git add .
   git commit -m "Initial commit"
   git push -u origin main
   ```

4. **Прислать ссылку** в чат группы (раздел «Обучение → Домашка»)
