## Задание 1: Установить Claude Code ✅

### Шаги:
1. Убедитесь, что **подписка Claude** активна (Pro или Max)
2. Откройте **новый терминал** в Antigravity
3. Выполните команды:

```powershell
npm install -g @anthropic-ai/claude-code
claude
```

4. Выберите **вариант 1** (подписка) при авторизации
5. Войдите под аккаунтом, где оплачена подписка
6. Проверьте: набрав `/model` → убедитесь, что стоит **Opus 4.6**

### Настройка:
- Панель терминала → ПКМ → **Move Panel → Right**
- Так Claude Code будет удобно рядом с чатом Antigravity

---

## Задание 2: Установить Skill Creator

1. В Claude Code набрать `/plugins`
2. Перейти в **Discover** → поиск → `skill`
3. Выбрать **Skill Creator**
4. Установить **глобально** (User scope)
5. **Перезапустить терминал** (обязательно!)
6. Проверить: `/skill-creator` должен появиться в меню

---

## Задание 3: Создать первый скилл

1. В Claude Code вызвать `/skill-creator`
2. Описать задачу: **«Создай скилл для качественной проверки кода»**
3. Когда спросит детали — ответить: **«Сделай на своё усмотрение, include тесты»**
4. Дождаться, пока он создаст и протестирует несколько вариантов
5. Выбрать лучший скилл по его рекомендации

---

## Задание 4: Проверить проект с помощью Claude Code

1. Открыть свой **текущий рабочий проект** в Antigravity
2. В терминале запустить `claude`
3. Написать: **«Изучи весь проект и расскажи, про что он»**
4. Затем: **«Используя скилл review, проверь код проекта и дай обратную связь»**
5. Скопировать результат → вставить в **чат Antigravity** → попросить исправить найденные ошибки

---

## Задание 5 (бонус): Добавить скилл с маркетплейса

1. Перейти на сайт с готовыми скиллами (ссылки из чата группы)
2. Найти интересный скилл (например, **frontend** или **code review**)
3. Скопировать команду установки
4. Выполнить в Claude Code
5. Проверить, что скилл работает

---

## 💡 Советы

> **Не забывайте:** Claude Code **безвозвратно** меняет файлы. Если хотите просто спросить — начинайте с фразы: «Не пиши код, объясни мне...»

> **Токены:** На тарифе Pro лимиты ограничены. Можно переключиться на Sonnet (`/model` → Sonnet), чтобы экономить.

> **Практика:** Попробуйте использовать Claude Code как **аудитора** для проекта, который вы разрабатываете в Antigravity.
