## Задание 1: Улучшить дизайн вашего сайта ⭐ (основное)

Возьмите сайт/приложение, которое вы создали ранее, и улучшите его дизайн, используя подходы из лекции:

### Шаги:
1. Соберите **2–3 референса** (Pinterest, 21st.dev или понравившиеся сайты)
2. Попросите ИИ **изучить ваш проект** и его функционал:
   ```
   Изучи проект, весь функционал и расскажи,
   что может этот сервис и для какой аудитории.
   ```
3. Проработайте **структуру лендинга** (хедер → hero → преимущества → фичи → демо → футер)
4. Попросите написать текст по **JTBD** (Jobs to be Done):
   ```
   Напиши тексты для главной страницы по методологии Jobs to be Done
   ```
5. Закиньте референсы и дайте финальный промпт:
   ```
   Сделай иерархию акцентов. Соблюдай все правила UX и UI.
   ```

### Критерии:
- Сайт **не должен быть шаблонным** — добавьте уникальные элементы
- Должна быть **иерархия акцентов** (жирный текст, фокус внимания)
- Должен быть **адаптивен** (мобильная версия)

## Задание 2: Попробовать Google AI Studio

1. Зайдите в **Google AI Studio** под аккаунтом Ultra
2. Выберите фреймворк **Next.js**
3. Опишите дизайн вашего сайта или дайте текст + референсы
4. Сравните результат с тем, что делает Antigravity
5. Если результат Google AI Studio лучше — **скачайте код** и закиньте в Antigravity

### Перенос из Google AI Studio:
```
Вот я разработал лендинг в Google AI Studio.
Замени текущую главную страницу на этот дизайн и запусти.
```

## Задание 3: Работа с ветками GitHub

1. Откройте ваш **репозиторий** на GitHub
2. Попросите Antigravity **создать новую ветку**:
   ```
   Создай отдельную ветку в репозитории, назови её "design"
   ```
3. Внесите изменения в дизайн в этой ветке
4. **Отправьте изменения** в ветку:
   ```
   Отправь код в ветку design
   ```
5. Проверьте на GitHub, что ветка появилась и содержит изменения

## 💡 Советы

> **Иерархия акцентов** — главный принцип. Где подсветили — там внимание. Используйте жирный текст, контрастные цвета.

> **Не полагайтесь только на ИИ** — используйте референсы, 21st.dev, Pinterest. Уникальность = ваш выбор элементов.

> **UX важнее UI:** Сначала удобство, потом красота. Убеждает логика.

> **Комбинируйте инструменты:** Antigravity для кода, Google AI Studio для фронтенда, Pinterest для референсов.
